# 项目使用说明
===========================
1. 进入server目录，安装相关包
```
npm install
```
2. 然后在server目录中启动3001端口
```
npm run start
```
3. 再进入client目录中，安装相关包
```
npm install
```
4. 在client目录中启动3000端口
```
npm run start
```
5. 在client目录中打包（我是用webpack将其打包在了server的public目录中）
```
npm run build
```
6. 直接打开"http://localhost:3001/index.html"
